# <a name="prerequisites-for-wi-fi-and-vpn-profiles-in-configuration-manager"></a>Предварительные требования для профилей Wi-Fi и VPN в Configuration Manager

*Область применения: Configuration Manager (Current Branch)*

Профили Wi-Fi и VPN в Configuration Manager имеют только зависимости в пределах продукта.

Для управления параметрами доступа к ресурсам компании, такими как профили сертификатов, профили Wi-Fi и профили VPN, необходимо иметь следующие разрешения системы безопасности:  

- Для просмотра и контроля оповещений и отчетов по Wi-Fi и профилям: **Создание**, **Удаление**, **Изменение**, **Изменение отчета**, **Чтение** и **Запуск отчета** для объекта **Оповещения**.  

- Для создания профилей сертификатов и управления ими: **Политика автора**, **Изменение отчета**, **Чтение** и **Запуск отчета** для объекта **Профиль сертификата**.  

- Для управления развертываниями профилей Wi-Fi, сертификатов и VPN: **Развертывание политик конфигурации**, **Изменение оповещения о состоянии клиента**, **Чтение** и **Чтение ресурса** для объекта **Коллекция**.  

- Для управления всеми политиками конфигураций: **Создание**, **Удаление**, **Изменение**, **Чтение** и **Настройка области безопасности** для объекта **Политика конфигурации**.  

- Для выполнения запросов, связанных с профилями VPN и Wi-Fi: разрешение на **Чтение** для объекта **Запрос**.  

- Для просмотра сведений о профилях Wi-Fi и VPN в консоли Configuration Manager: разрешение на **Чтение** для объекта **Сайт**.  

- Для просмотра сообщений о состоянии по профилям VPN и Wi-Fi: разрешение на **Чтение** для объекта **Сообщения о состоянии**.  

- Для создания и изменения профиля сертификата доверенного ЦС: **Политика автора**, **Изменение отчета**, **Чтение** и **Запуск отчета** для объекта **Профиль сертификата доверенного ЦС**.  

- Для создания профилей VPN и управления ими: **Политика автора**, **Изменение отчета**, **Чтение** и **Запуск отчета** для объекта **Профиль VPN**.  

- Для создания профилей Wi-Fi и управления ими: **Политика автора**, **Изменение отчета**, **Чтение** и **Запуск отчета** для объекта **Профиль Wi-Fi**.  

Роль безопасности **Диспетчер доступа к ресурсам компании** включает все разрешения, необходимые для управления профилями Wi-Fi в Configuration Manager. Дополнительные сведения см. в статье о [настройке безопасности](../../core/plan-design/security/configure-security.md).
