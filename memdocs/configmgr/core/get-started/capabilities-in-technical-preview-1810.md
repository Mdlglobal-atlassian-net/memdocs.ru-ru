# <a name="capabilities-in-configuration-manager-technical-preview-version-1810"></a>Возможности Configuration Manager версии Technical Preview 1810 

*Область применения: Configuration Manager (ветвь Technical Preview)*

В этой статье содержатся сведения о функциях, доступных в Configuration Manager версии Technical Preview 1810. Установите эту версию, чтобы обновить и добавить функции в сайт версии Technical Preview.
