# <a name="product-end-of-support"></a>Прекращение поддержки продукта

*Область применения: System Center Configuration Manager 2007, Forefront Endpoint Protection 2010*

## <a name="system-center-configuration-manager-2007-and-forefront-endpoint-protection-2010-are-out-of-support"></a>Прекращение поддержки **System Center Configuration Manager 2007** и **Forefront Endpoint Protection 2010**

Поддержка **System Center Configuration Manager 2007** и **Forefront Endpoint Protection 2010** прекращена с 9 июля 2019 г. Корпорация Майкрософт больше не предоставляет для этих продуктов следующие услуги:

- Обновления для системы безопасности
- Обновления, не связанные с безопасностью
- Возможности технической поддержки (бесплатные или платные)
- Обновления технического содержимого в Интернете

Корпорация Майкрософт рекомендует воспользоваться следующим планом миграции.

- Миграция Configuration Manager 2007 в Configuration Manager (Current Branch).  

- Миграция Forefront Endpoint Protection 2010, интегрированного с Configuration Manager 2007, в Configuration Manager (Current Branch), в состав которого входит решение Endpoint Protection.  

- Миграция автономного решения Forefront Endpoint Protection 2010 в автономное решение System Center Endpoint Protection (Current Branch).
